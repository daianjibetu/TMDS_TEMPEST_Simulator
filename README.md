# TMDS_TEMPEST_Simulator
HDMI TEMPEST 再構築画像シミュレータ

TMDS信号については[こちら](https://github.com/daianjibetu/TMDS_Signal_Simulator)を参照  

画像をアップロードすると、グレースケールに変換された後、画像の再構築が行われます。
